Blind Code

~ Code in the dark, trust your instincts—can you crack the challenge without sight? ~

Description

Participants must code a random problem without a monitor for 20 minutes. A lifeline allows them to briefly view their screen, but they can't use the keyboard during this time—error-free code wins the cash prize.

Rules

Instructions to be followed:

Round 1: Prelims           Round 2: Finals

Welcome to Blind Code, where your coding instincts take center stage! With no monitor to guide you, can you craft a flawless solution in just 20 minutes? Use your lifeline wisely, and code your way to the cash prize!

• Player will be provided with a random coding problem

• Players are asked to code without a monitor

• Players will be provided with a lifeline in between the event which they can use   to switch on the monitor

• Players can't use the keyboard while using the lifeline
